# Twanano's 19th Birthday 💕

A neon-themed browser experience built for a birthday celebration. Users follow a sequence of flashing piano keys to unlock an audio beat and reveal a final birthday blessing.

## Files

- `index.html` - Main page structure and audio/player layout.
- `style.css` - Neon-inspired styling and animation for the piano interface.
- `script.js` - Interactive sequence logic, sound playback, confetti effects, and end-screen reveal.
- `beat.mp3` - Background audio track played when the sequence is completed.
- `baby.jpeg` - Final celebration image shown on completion.

## How to use

1. Open `index.html` in a modern browser.
2. Follow the flashing highlighted piano keys.
3. Click the correct key for each step.
4. After all 19 steps are completed, the beat plays and the final birthday message appears.

## Notes

- The target key sequence is generated randomly each time the page loads.
- Sound playback uses the browser `AudioContext` and oscillator tones for instant key feedback.
- The final screen includes a reset button that reloads the page.
